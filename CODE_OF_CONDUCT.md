# Code of Conduct

## 我们的承诺

为了营造开放、友善、尊重的社区环境，参与 Everything-Windsurf-Codex / EWC 的每个人都应以建设性的方式交流。

无论经验水平、使用工具、语言背景、身份或观点差异如何，我们都欢迎善意讨论和高质量反馈。

## 鼓励的行为

- **尊重他人**：针对问题和方案讨论，不攻击个人。
- **清晰表达**：说明问题、背景、复现步骤和期望结果。
- **接受差异**：允许不同工具偏好，例如 Windsurf、Codex、Claude Code 或其他 AI coding 工具。
- **建设性反馈**：提出问题时尽量给出改进建议。
- **保护隐私**：不要公开他人的 token、账号、日志、私有路径或敏感信息。

## 不接受的行为

- 人身攻击、骚扰、歧视、嘲讽或恶意挑衅。
- 公开他人的隐私、凭据、私有日志或敏感信息。
- 故意提交破坏性 PR、恶意脚本或误导性内容。
- 在讨论中持续偏离主题或重复制造冲突。
- 要求维护者接受超出项目定位的修改。

## 处理方式

维护者可以根据情况采取以下措施：

- 提醒讨论回到技术问题。
- 编辑或隐藏不合适的评论。
- 关闭无建设性 Issue 或 PR。
- 限制持续违反规则者的参与。

## 项目定位提醒

EWC 优先服务于 Windsurf + Cascade + Codex MCP/CLI fallback 的协作模式。欢迎不同意见，但项目不会因为单一偏好而放弃其核心定位。
